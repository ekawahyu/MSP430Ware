DEPRECATED - This function configures the MSP430 Port Mapper.

This function port maps a set of pins to a new set.

Parameters
baseAddressis the base address of the PMAP control module.
portMappingis the pointer to init Data
PxMAPyis the pointer start of first PMAP to initialize
numberOfPortsis the number of Ports to initialize
portMapReconfigureis used to enable/disable reconfiguration Valid values are:
  • PMAP_ENABLE_RECONFIGURATION
  • PMAP_DISABLE_RECONFIGURATION [Default]

Modified bits of PMAPKETID register and bits of PMAPCTL register.

Returns
None
