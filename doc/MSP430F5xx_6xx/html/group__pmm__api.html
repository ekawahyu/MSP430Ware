Modified bits of PMMCTL0 register, bits of PMMIFG register, bits of PMMRIE register, bits of SVSMHCTL register and bits of SVSMLCTL register.

Returns

Returns interrupt status.

Parameters
maskis the mask for specifying the required flag Mask value is the logical OR of any of the following:
  • PMM_SVSMLDLYIFG
  • PMM_SVMLIFG
  • PMM_SVMLVLRIFG
  • PMM_SVSMHDLYIFG
  • PMM_SVMHIFG
  • PMM_SVMHVLRIFG

  • PMM_PMMRSTIFG
  • PMM_PMMPORIFG
  • PMM_SVSHIFG
  • PMM_SVSLIFG
  • PMM_PMMLPM5IFG
Returns
Logical OR of any of the following:
  • PMM_SVSMLDLYIFG
  • PMM_SVMLIFG
  • PMM_SVMLVLRIFG
  • PMM_SVSMHDLYIFG
  • PMM_SVMHIFG
  • PMM_SVMHVLRIFG
  • PMM_PMMBORIFG

  • PMM_PMMPORIFG
  • PMM_SVSHIFG
  • PMM_SVSLIFG

    indicating the status of the masked interrupts
