void EUSCI_I2C_masterInit ( uint16_t  baseAddress,
uint8_t  selectClockSource,
uint32_t  i2cClk,
uint32_t  dataRate,
uint8_t  byteCounterThreshold,
uint8_t  autoSTOPGeneration 
)

DEPRECATED - Initializes the I2C Master block.

This function initializes operation of the I2C Master block. Upon successful initialization of the I2C block, this function will have set the bus speed for the master; however I2C module is still disabled till EUSCI_I2C_enable is invoked.

Parameters
baseAddressis the base address of the I2C Master module.
selectClockSourceis the clocksource. Valid values are:
  • EUSCI_I2C_CLOCKSOURCE_ACLK
  • EUSCI_I2C_CLOCKSOURCE_SMCLK
i2cClkis the rate of the clock supplied to the I2C module (the frequency in Hz of the clock source specified in selectClockSource).
dataRatesetup for selecting data transfer rate. Valid values are:
  • EUSCI_I2C_SET_DATA_RATE_400KBPS
  • EUSCI_I2C_SET_DATA_RATE_100KBPS
byteCounterThresholdsets threshold for automatic STOP or UCSTPIFG
autoSTOPGenerationsets up the STOP condition generation. Valid values are:
  • EUSCI_I2C_NO_AUTO_STOP
  • EUSCI_I2C_SET_BYTECOUNT_THRESHOLD_FLAG
  • EUSCI_I2C_SEND_STOP_AUTOMATICALLY_ON_BYTECOUNT_THRESHOLD
Returns
None

  • EUSCI_I2C_OWN_ADDRESS_OFFSET2
  • EUSCI_I2C_OWN_ADDRESS_OFFSET3

void EUSCI_I2C_enableInterrupt ( uint16_t  baseAddress,
uint16_t  mask 
)

Enables individual I2C interrupt sources.

Enables the indicated I2C interrupt sources. Only the sources that are enabled can be reflected to the processor interrupt; disabled sources have no effect on the processor.

Parameters
baseAddressis the base address of the I2C module.
maskis the bit mask of the interrupt sources to be enabled. Mask value is the logical OR of any of the following:
  • EUSCI_I2C_NAK_INTERRUPT - Not-acknowledge interrupt
  • EUSCI_I2C_ARBITRATIONLOST_INTERRUPT - Arbitration lost interrupt
  • EUSCI_I2C_STOP_INTERRUPT - STOP condition interrupt
  • EUSCI_I2C_START_INTERRUPT - START condition interrupt
  • EUSCI_I2C_TRANSMIT_INTERRUPT0 - Transmit interrupt0
  • EUSCI_I2C_TRANSMIT_INTERRUPT1 - Transmit interrupt1
  • EUSCI_I2C_TRANSMIT_INTERRUPT2 - Transmit interrupt2
  • EUSCI_I2C_TRANSMIT_INTERRUPT3 - Transmit interrupt3
  • EUSCI_I2C_RECEIVE_INTERRUPT0 - Receive interrupt0
  • EUSCI_I2C_RECEIVE_INTERRUPT1 - Receive interrupt1
  • EUSCI_I2C_RECEIVE_INTERRUPT2 - Receive interrupt2
  • EUSCI_I2C_RECEIVE_INTERRUPT3 - Receive interrupt3
  • EUSCI_I2C_BIT9_POSITION_INTERRUPT - Bit position 9 interrupt
  • EUSCI_I2C_CLOCK_LOW_TIMEOUT_INTERRUPT - Clock low timeout interrupt enable
  • EUSCI_I2C_BYTE_COUNTER_INTERRUPT - Byte counter interrupt enable

Modified bits of UCBxIE register.

Returns
None
